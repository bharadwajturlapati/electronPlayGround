## Installation
Setup :<br />
1. Add the following lines in exclude of .git<br />
node_modules<br />
ts/**/*.js<br />
less/**/*.less<br />
.tscache<br />
index.js<br />
utils/.idea<br />
2. To install node modules run the following command [npm install]<br />
3. To run the project run the following command [npm start]<br />
